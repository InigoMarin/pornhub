# PornHub  Video play , The UNIX way

pornhub is a console search and view video program.
It install two programs:

+ pornhub.py
+ pornhub : Bash script to pornhub and pipe programs

## Installation

`sudo make`

## Uninstall

`sudo make clean`

## Program Requeriments

+ **fzf**
+ **mpv**
+ **xargs**

## Python pip requerimeints

+ **pornhub-api**


## Usage

`pornhub <textsearch>`
